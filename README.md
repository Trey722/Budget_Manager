# Budget Manager

## Overview
CLI-based budget tracker

## Features
- Record and track expenses
- View summary of expenses
- Manage budget categories
- Generate reports
- Connect to API's 
- Export Data
- Import data

## Installation

'''bash
  git clone https://github.com/Trey722/Budget_Manager.git
'''

## Usage
1. Navigate to the project directory.
2. Run the CLI:
    ```bash
    python3 CLI.py
    ```
3. With python
    ```bash
    python CLI.py
    ```
3. Log in or sign up using username and password 

budget manager.

## Sample Commands

- To add an expense or income:
1. Use the guide
    ```bash
    budget add 
    ```
2. Do it yourself 
    ```bash
        budget add --type "type" --amount "#" --category "section" --description "" --date "YYYY-MM-DD" "descritipon" --instituion ""
    ```

- to create a budget 
    ```bash
        create --budget
    ```




- To view summary:
    ```bash
    budget summary
    ```
